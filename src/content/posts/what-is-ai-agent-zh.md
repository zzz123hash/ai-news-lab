## AI Agent 的简单定义

AI Agent 可以理解为能够围绕一个目标，自主规划步骤、调用工具并根据结果继续行动的 AI 系统。它不只是回答问题，而是尝试完成任务。

例如，你让一个 Agent 整理竞品信息，它可能会搜索资料、读取网页、提取要点、生成表格，最后给出结论。

## Agent 和聊天机器人的区别

普通聊天机器人主要完成一次问答。Agent 更强调连续行动和工具使用。

常见差异包括：

- 是否能拆解任务
- 是否能调用外部工具
- 是否能根据中间结果调整计划
- 是否能输出可执行结果

## 目前的限制

Agent 并不等于完全自动化。它可能误解目标、调用错误工具，或在信息不足时给出看似合理但不准确的结论。

实际使用时，仍需要权限控制、日志记录和人工审核。
